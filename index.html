How can I participate?

  1. Read the rules
  2. Sign up to the project
  3. Install Hackatime (tutorial HERE)
  4. Develop your project for at least 30 hours (remember to commit on GitHub frequently!)
  5. Upload your project to GitHub, adding a description and including a short video where you showcase it
  6. Share the GitHub link in the #hackberry-ship channel on Slack
  7. Enter your shipping details HERE

Rules!

    NO Autokey or time cheat (fraud will ban you from all Hack Club !)

    NO Copy & paste of other people's project

    NO unmodified old projects (you can use already started projects, but you need to specify what you have added during the program) or used in other HC programs (except SOM*)
    SUMMER OF MAKING RULES: If your project is already registered, you must specify it in the #hackberry-ship message, also adding a description/list of the things added during Hackberry. Without it, your project will not be accepted.

    YES, you can use tutorials, but you must add some personalization

    YES, you can use AI, but only as a support!
